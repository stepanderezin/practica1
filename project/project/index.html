<3 магазин повседневной одежды для гиков <3

    Компания ориентирована на гиков, и людей из IT сферы, представленная одежда мягкая, приятная на ощупь, удобная для того чтобы несколько часов подряд сидеть за компьютером, работая или играя в игры. Ассортимент в основном представлен в виде худи, кофт, свитеров, пижамы, штанов и прочей одежды. Так же представлены различные аксессуары и прочие товары: чехлы для телефонов, повязки, подушки для шеи, мягкие игрушки и мини-фигурки с маскотом компании.
